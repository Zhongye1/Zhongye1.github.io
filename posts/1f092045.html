三月

发表于 2025-03-05|更新于2025-04-05|随记
|字数总计:2065|阅读时长:6分钟|阅读量:

3月1日

无所事事

遂搭公交车前往黄埔

上次看到这么大片的油菜花还是在湖北

等天气好了找个时间再来看看

3月2日

搭校园论坛

薅的导师服务器

反正没干正事

3月3日

部署了校园论坛 GZHU聊天室

网站开源的一套部署下来

无非前端npm run build 编译页面后端pm2运行然后搞nginx配置再弄个配置ssl配置后端配置什么的有点头疼，被websocket的问题卡了一会

这个论坛呢无非凉了或者火了被导师发现薅服务器羊毛被rm -rf反正域名薅的服务器也是薅的成本接近为0，只参与部署没碰开发算法逻辑打算同校找开发同行加入别人开发团队别人问技术实现什么的也答不上来然后被祝你幸福晚上瞎捣鼓熬夜白天又犯困做梦被数据结构追最后早八上第二天课然后论坛什么的拉点人来活跃了几天就凉在之后没人记得了

开发什么的还是得学

要头疼的事情太多了，而且大学教的东西太少，开发这块更少

还得靠自己

3月4日

至少研究那么久跟老师做的项目有头绪了，算是搞完了一个阶段

晚上刷力扣

怀疑上学期学了个假的python，第二题就做不出来上学期怎么期末八十多的

还要练，到时候整理个笔记

大一没好好学数据结构算法给自己挖的坑还得填

3月5日

广东还有倒春寒的…

反正下午哪都不想去

这种天气就适合睡觉的说

傍晚码博客

该摸鱼时就摸鱼

明天没早八

3月8日

被导师的新需求吓晕

还以为本科跟项目没啥问题

怎么要求越来越离谱（晕

问题是都还要学的太多了

3月9日

闲来无事整理笔记

又水了几篇博客

https://zhongye1.github.io/posts/a12f4d8d.html

（概率论
https://zhongye1.github.io/posts/ec1ceb96.html

（常微分

这周末约等于摆烂

下周末最好跑掉，受不了二手烟

最近发生挺多事，感觉现状不理想，以后也难办

有一种把路走死了的感觉

也不见得能回2023把自己揍一顿

反正我想如果当初有机会的话现在会不会在那个校园能接触更多开发同好和交流资源什么的，还能有一堆开源社群或者linux user group之类的同好

而不是学校里周围一堆现充天天奶茶团建loopy表情包或者抽烟深情电竞窝宿舍高手

为什么我在大学的人脉连几个上得了github的都拉不到

我真感觉我把路走死了

之后也难办

努力搞出去罢

我真的服了看我不把这些破事凹死

我还不信了，妈的

最近回暖了

春夏秋冬

这些是不是还会再来一轮

可惜往后的一年压力会不会更大还说不定

感觉2024还没过完就是2025了

然而之前的许多我还是放不下

第一年一塌糊涂真的是

然后第二年也一堆事

第三年第四年呢

为什么就感觉周围的生活没有光

为啥是去南漂，一年就老家最没机会回去

为啥刚好在高考出意外，直接跑到一个填志愿时才知道的地方读书？

为啥周围人刚好是自己的人际关系一塌糊涂，烂到一个能拉上开发的都没

为啥刚好在自己面临这些时周围刚好缺资源自己缺技术学校缺环境

怎么这些事情就直接直接挨到我头上了

怎么偏偏就是这样能把路给走死了

天天焦虑结果啥都干不成然后又废掉两天



反正这个博客被三大运营商墙了别人看不到

发牢骚也没人看了，好极了



假如说这些问题一直放在这里无法解决，那怎么办呢，那它会解决吗

所以这就让人头疼了

不管是认输还是一头撞死在上面问题还是没有改变

草了，真够麻烦的



过一段时间后这些也不太可能有人记得

把什么东西保存下来的话

其实会很难的吧



熬不了夜先睡了

我一想也许我整理笔记到博客也不太会是因为闲来无事

3月10日

所以说今天睡了一下午

啥都没干啊（恼

明天得做点事了

3月11日

想在学校组一个开源组织，搭个论坛和镜像站什么的

不过怕实力不够搞不起来（悲）

3月13日

写网页

翻论坛

3月15日

写完基础网页然后写中心度查询

上一周课燃尽了这学期还喜提单休

导师画的网页设计图

3月16日

周日前往珠江出海口看海

广州也有自己的江之岛

就是交通不太方便

晚上搞完网页了，建了个仓库

neo4j-node-viewer

基于neovis.js的一个以太坊网络数据展示用前端

📅 Updated: 2025-03-16
JavaScript Neo4j Neovis WebSocket

3月19日

又鸽了几天

开学快一个月了好像还是没做什么

网页源码给导师后也没啥消息了，今天试着联系下

和博友讨论组织的事，结果是没搞头

当初去了广工就好了

早就想到会这样，之前也问过别人，回复是说当前的学历不支持这些想法

我想这是不是太消极了一点，不过现在这个情况结果确实不意外

也许吧

至少自己要走出去

想了很多又删了，发牢骚没意思

不管怎么选择都会后悔的，然而我还是想回23年把做选择的自己干死

比把自己交代在这里会好点

3月27日

搭环境搞容器建集群

反正没搞定

反正就是大战命令行

3月28日

入夏几天后回寒

这段时间事情太多

然后之后也难搞

夜降温下雨，下晚课后想去江边走走

不过没吃晚饭，遂放弃

这周末也有挺多事要做，至少抽出时间来去江边走一趟

关于未来

没想法，能力不够，还得学

反正焦虑不能当饭吃，该刷的还要刷

之后怎么说，难搞

以后可能考研或者找工作还是什么地方也说不定

不知道会不会很惨，反正打死不干销售

在这天的最后一段时间，遂想到

也许平等不会像现在这样分三六九等

但总所周知的，现在这样的，

也做不出什么。

经常犯困，还是早点睡吧

啊，周末事情也好多

其实经常熬夜是因为白天时间留给自己支配的没有多少

估计等我扛不住了这个恶习才能改

每次都这样

这些想法，谁会在意呢

反正也不太可能有人找到这里来

也不知道以后自己看到这些会怎么想

在这些日子里拼尽全力无论如何呐喊也没人会听到这些故事

若是不记下来的话自己也会遗忘

关于以后，遥遥无期了

至少，离去的日子里不要太悲伤

3月31日

31号了，那么三月也是过完了

月底在珠江边走了一趟

这几天来寒潮，天气挺冷，

想到过几天回暖入夏又有些伤感

也是，就是到四月了

于是又要开个帖来讲述这些日子里的无聊生活以及牢骚

斯人已去，剩下的这个四月又会有什么呢

再看吧

活在当下是最难
