# 数据可视化的常见图表

本文主要介绍数据可视化的常见图表，主要包括其基础信息、构成与视觉通道、适用场景、优缺点和常见变体

# 一、比较与排序图表

### 柱状图

<img src="/数据可视化/bar-chart.png" width="50%">

- **基础信息**：以矩形高度为测度的统计图，比较两个及以上类别的定量指标；由横纵轴、矩形柱、图例组成，可纵向或横向。
- **视觉通道**：类别→颜色+位置；数值→长度。横轴放定类/定序/定距数据，纵轴放定量数据。
- **适用场景**：分类数据的对比与排序，堆叠可同时看数值变化与占比；类别过多时不适用。
- **注意**：纵轴须从 0 起；柱间距约为柱宽一半；同组同色系、强调用对比色；多系列按逻辑排序
- **优缺点**：优点是分类对比直观；缺点是类别多则标签拥挤、不擅长表达关系/层级/分布、依赖坐标轴设置易误导。
- **变体**：横向条形图（类别多时适用）、径向条形图（极坐标、空间利用率高）、双向柱状图（正反对比）、象形柱状图（具象语义）、3D 柱状图

### 环柱状图

<img src="/数据可视化/circular-bar-chart.png" width="50%">

- **基础信息**：柱状图沿圆周弯曲，柱子变成绕圆心的弧形条，紧凑美观。
- **视觉通道**：类别→颜色+位置；数值→面积（弧长+半径）。精度低于长度，不宜精确对比（玫瑰图用长度，性质不同）。
- **适用场景**：类别多、想省横向空间且重设计感，如排行榜、活动榜单。
- **注意**：内圆半径别太短；不宜表现差距过大或过于接近的数据；纵轴尽量统一；半径平方关系会夸大差异；数据太少改用玫瑰图/柱状图；复杂比较不宜用极坐标。
- **变体**：环形柱状堆叠图、玫瑰图（南丁格尔图）、半圆柱状图。

### 子弹图

<img src="/数据可视化/bullet-chart.png" width="50%">

- **基础信息**：狭小空间内同时展示实际值、目标值与评价区间，取代仪表盘式表达。
- **视觉通道**：类别→颜色视觉通道（背景色条）；数值→坐标轴位置（数据条、目标刻度）。
- **适用场景**：仪表盘、KPI 监控，业绩完成进度，判断指标是否达标；适用不超过 5 条定量数据、2–5 个定性区间。
- **注意**：标签按从大到小、从左到右排列；主标签用纯黑、刻度用浅灰；定性区间 3–5 个为宜；主标记一般垂直居中
- **优缺点**：优点是空间小信息密度高，缺点是有学习成本。
- **变体**：反向子弹图（负面值）、层叠子弹图（阶段数据）

### 哑铃图

<img src="/数据可视化/dumbbell-chart.png" width="50%">

- **基础信息**：一条线连两个圆点，表示同一对象在两种状态下的数值，形似哑铃。
- **视觉通道**：区分高低值→颜色+位置；比较数值大小→位置；比较极差→长度。水平轴定量、垂直轴定性。
- **适用场景**：前后变化或两组对比，如改革前后、男女差异、去年与今年、外界刺激后的变化。
- **注意**：数据边界差异大而端点差异小时杠杆过短不易观察，需颜色辅助；只适合比较起止两点；数据过多过密则失效，变化幅度难精确区分时改用双折线图或瀑布图。
- **变体**：火柴图、纵向哑铃图、多变量哑铃图、甘特图。

### 马赛克图

<img src="/数据可视化/mosaic-chart.png" width="50%">

- **构成与视觉通道**：纵横轴 + 色块 + 图例，表现二维数据的比较关系。
- **适用场景**：二维分类变量比较、边际分析。

### 雷达图

<img src="/数据可视化/radar-chart.png" width="50%">

- **基础信息**：多变量数据沿中心发散的多轴排布，各指标连成闭合多边形。
- **视觉通道**：类别→位置；性能比较→面积（形状）。
- **适用场景**：少数对象的多维综合评价与对比，如能力模型、产品多属性；适用 5 组以内、不超过 30 维。
- **注意**：组数和变量都不宜过多（多边形/坐标轴过密、上层遮挡下层）；数值需先标准化以便同级比较；不擅长精确比较；形状面积随值的平方增长，会夸大微小差异。
- **变体**：填充雷达图、极坐标图（含南丁格尔玫瑰图）。

### 平行坐标图

<img src="/数据可视化/parallel-coordinates.png" width="50%">

- **基础信息**：一组平行纵轴表示不同变量，每条记录在各轴取值连成折线，用于高维/多元数据。
- **视觉通道**：类别→颜色；变量在某属性下的表现→位置。Y 轴定序、X 轴属性类别、颜色区分数据种类。
- **适用场景**：探索多变量关系、发现聚类与异常。
- **注意**：颜色不宜多以免线条混乱；适当增大轴间距提升可分辨性、必要时缩放；保证线条归属清晰；各轴量纲可能不同，需标准化。
- **变体**：三维平行坐标图、星形平行坐标图。

### 词云图

<img src="/数据可视化/word-cloud.png" width="50%">

- **基础信息**：词语按频率以不同字号、颜色排布，字号越大权重越高。
- **视觉通道**：颜色→颜色视觉通道；大小→面积；图形/区域图片→形状；位置→位置。文本内容定类、大小定比、颜色定量。
- **适用场景**：快速展示高频关键词、舆情与主题热点的概览传播。
- **注意**：文字不可重叠；选与主题相关的轮廓图；个别权值过大时取 log 压缩差距；文本太少或权重区分不明显则效果差；面积难感知精确数值、长词易造成视觉误差；不绘无意义高频词。
- **变体**：形状词云、分类着色词云。

## 图表特性对比

| 维度           | 表现                                                                                                                                                |
| -------------- | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| **数据量大**   | 平行坐标图、词云图、环柱状图能承载大量数据；柱状图、哑铃图、雷达图数据多时可读性明显下降                                                            |
| **信息精确性** | 高：子弹图、柱状图、哑铃图（依托坐标轴）；低：词云图（面积/颜色）、平行坐标图与雷达图（轴常被统一度量）、环柱状图（极坐标失去水平参考、半径反效果） |
| **传递效率**   | 高效表现数值：哑铃图、柱状图、环柱状图；词云图靠面积/颜色直观看关键词；雷达图、子弹图、平行坐标图维度多，比较时耗时更长                             |

# 二、局部与整体可视化图像

与比较排序可视化图形不同，整体与局部可视化图形重点关注局部占整体的比重，有时也被称为占比类可视化图形，更多使用面积来表示整体与局部的关系

很多可视化图形可同时展示比较于排序以及整体与局部的关系，比如矩形树图

整体与局部可视化图形可以看做对定量树形层级关系的一种可视化，可以分为两层与多层，单路径和多路径

**局部与整体图形的优势与局限性**

| 图表         | 优势                       | 局限性                     |
| :----------- | :------------------------- | :------------------------- |
| **维恩图**   | 展示相交关系               | 仅适合少量集合             |
| **饼图**     | 简单易懂的占比显示         | 分类过多效果差，信息少     |
| **环形图**   | 显示文本信息，支持多环环比 | 需避免过度嵌套             |
| **旭日图**   | 多层数据比例显示           | 复杂度随层级增加           |
| **圆锥树图** | 分类型数据一目了然         | 不宜过于细节，存在空间留白 |
| **矩形树图** | 空间利用率高               | 级别过多不可准确读         |
| **漏斗图**   | 比较各业务流程             | 更适用于流程数据           |

### 饼图

<img src="/数据可视化/pie-chart.png" width="50%">

- **基础信息**：把一个圆按各部分占比切成扇形，表现部分占整体的比例。

- **视觉通道**：类别→颜色视觉通道；占比→面积，形状，角度

- **适用场景**：2-9个类别占比情况，如市场份额、预算分配。

- **注意**：分类不宜超过五六块（多的较小的标为其他）、份额最大的放12点方向，不要用3d，分割类别表示强调，排序并标注百分比
- **优缺点**：优点是占比直观，缺点是切片多难读、不适合精确比较
- **变体**：分割饼图（强调），华夫饼图（识别小比例差异）、斯贝图（饼图加玫瑰图）、玫瑰图、双层饼图（重属关系）、复合饼图（复杂数据）

### 环形图

<img src="/数据可视化/donut-chart.png" width="50%">

- **基础信息**：中间挖空的饼图，中心可放总数或标题。

- **视觉通道**：类别→颜色视觉通道；占比→形状，长度。

- **适用场景**：用途与饼图相同，更现代美观，常用于仪表盘和看板。

- **注意**：份额最大的放12点方向，分割类别表示强调，中间挖空面积要大
- **优缺点**：优点是美观、中心可复用，缺点同饼图
- **变体**：多层环形图、进度环形图、变形环形图

### 旭日图

<img src="/数据可视化/sunburst-chart.png" width="50%">

- **基础信息**：环形的层级结构图，从内到外逐圈表示数据层级关系（层级和归属）（原点越近层级越高）

- **视觉通道**：类别→颜色视觉通道；层级→同心环位置；占比→弧长。

- **适用场景**：带上下级归属的构成数据，如多级分类销售、目录占用。

- **注意**：颜色，饱和度区分分类，层级深或分类多时外圈细碎，需交互下钻
- **优缺点**：优点是同时表达占比与层级，缺点是层级多难读。
- **变体**：可下钻旭日图、带空值旭日图

### 圆堆积图

<img src="/数据可视化/circle-packing.png" width="50%">

- **基础信息**：用大圆套小圆表示包含关系，圆面积代表数值。

- **视觉通道**：类别→颜色；占比大小→面积；嵌套→位置。

- **适用场景**：表现层级结构和相对规模，如组织、分类、预算的嵌套构成。

- **注意**：层级不要太细，2到3个，表达精确性不建议用，分类占比太小难排布，不要选相差过大/过小的数据值，不难精确比较
- **变体**：气泡层级图、打包圆图。

### 矩形树图

<img src="/数据可视化/treemap.png" width="50%">

- **基础信息**：把整体区域递归切成矩形，面积表示数值，嵌套或相邻表示层级。

- **视觉通道**：类别→颜色视觉通道；占比→面积；层级→包含。

- **适用场景**：有限空间里展示大量带层级的占比数据，如磁盘占用、品类构成。

- **注意**：分类占比太小时难区分，颜色边框要设计好。不适合没有权重的数据，层级不要太多，按层级最高级别优先级排序
- **优缺点**：优点是空间利用率高、容量大，缺点是面积精度有限。
- **变体**：可下钻树图、多类别树图。

### 漏斗图

<img src="/数据可视化/funnel-chart.png" width="50%">

- **基础信息**：用自上而下逐级收窄的梯形条表示流程各阶段的数量递减。

- **视觉通道**：本流程不同环节-> 颜色；各环节留存数量->长度；各阶段转化趋势-> 角度（斜率）

- **适用场景**：分析转化流程流量分析，如从浏览到下单的各环节流失。商业分析，医疗分析

- **注意**：各阶段须有先后和递减逻辑
- **优缺点**：优点是流失一目了然、便于找瓶颈，缺点是只适合单向递减流程。
- **变体**：金字塔图、对称漏斗图。

# 三、分布类可视化图

在可视化中用于展示定量数据在其取值范围中的分布特征


**直方图与密度图的区别**

|              | **直方图**                     | **密度图**             |
| :----------- | :----------------------------- | :--------------------- |
| **纵坐标**   | 数据出现频数                   | 数据出现概率           |
| **横坐标**   | 数据区间                       | 数据取值               |
| **适用数据** | 离散和连续均可                 | 连续数据               |
| **表现形式** | 离散式                         | 连续式                 |
| **功能**     | 展示数据在不同区间内的分布情况 | 展示数据出现的概率分布 |

- 直方图适合单组数据，密度图可绘制多组（不超过5组）；
- 直方图样本量足够大；密度图具有足够数量；
- 直方图对样本数据直接利用，图形更贴合样本；密度图观察概率分布更准确美观

共同的局限性是？如果样本量不足，它们就无法准确反映数据的真实分布形态，容易产生误导性的结论

**小提琴图与密度图的区别**

- 小提琴图是箱线图和密度图的合体；
- 小提琴的纵轴代表数值，横轴代表不同属性；宽度代表出现的概率分布=密度图纵轴
  密度图的纵轴代表数据出现的频率，横轴代表具体数值
- 小提琴图可表示四分位数范围和置信区间；密度图可表示峰值、拐点；
- 小提琴图可表示多个属性，密度图表示多属性易引起混淆

> 可展示的数据组数：直方图 < 密度图 < 小提琴图
> 样本数据的贴合程度：小提琴图 < 密度图 < 直方图

既想要展示分布，又想更多组比较

### 直方图

<img src="/数据可视化/histogram.png" width="50%">

- **基础信息**：把连续变量切成相连区间，用矩形高度表示落入各区间的频数，表现分布。

- **视觉通道**：数值的组距，频数密度，频率→位置；频数→面积；

- **适用场景**：50条以上的定量数据，观察集中趋势、离散程度和形态，如成绩、年龄分布。

- **注意**：合适的组距（不能过大过小），区间宽度明显影响样貌，过宽掩盖细节、过窄杂乱。样本量>= 50 个 单个类别单个变量和柱状图比，宽度有意义
- **优缺点**：优点是分布形态直观，缺点是对分组敏感。
- **变体**：含拟合曲线直方图、镜像直方图、坐标变换直方图，分组直方图

### 密度图

<img src="/数据可视化/density-plot.png" width="50%">

- **基础信息**：直方图的平滑版本，用连续曲线估计概率分布，曲线下面积代表概率。

- **视觉通道**：类别-> 颜色； 取值范围-> 位置；种类，特征-> 形状 ；到x轴距离->长度

- **适用场景**：展示平滑分布形态、对比多组分布，如不同人群收入分布。

- **注意**：不超过4个，各曲线和横轴围起来面积为1，要连续变量，不适合起伏波动过大的，极值差异不能过大，重叠要用半透明。带宽越大曲线越平滑，样本大数据密小带宽，样本小数据不密大带宽
- **优缺点**：优点是平滑、便于叠加，缺点是纵轴是密度不直观。
- **变体**：核密度图、二维密度图、堆叠密度图。

### 箱线图 （多组数据分布特征比较）

<img src="/数据可视化/box-plot.png" width="50%">

- **基础信息**：用"箱子加须"概括最小值、下四分位、中位数、上四分位、最大值，箱外点为离群值。

- **视觉通道**：类别→颜色，位置；数据分布→长度+位置

- **适用场景**：比较多组数据的水平和离散度并发现异常，如各班成绩对比。看骗态分布，异常值集中较大值一侧右偏，较小值一侧左偏

- **注意**：不超过12组，不能提供骗态尾重精确度量，数据不宜过少。无序分组按中位数排列，上限和下限的计算，样本数据的选择
- **变体**：带数据点箱线图、分组箱线图、横向箱线图。

### 小提琴图（不同组连续数据异同分析，分布特征比较）

<img src="/数据可视化/violin-plot.png" width="50%">

- **基础信息**：箱线图与密度图结合，箱体两侧对称画出密度曲线，外形像小提琴。

- **视觉通道**：区间数值→位置；密度→长度；分组-> 颜色；数据量-> 面积

- **适用场景**：对比多组数据时同时关注分布形状，如不同处理组分布。每个月温度性差异分布

- **注意**：要用大数据量，只能比较分布情况没有准确概率密度信息，不能展示异常值
- **优缺点**：优点是信息丰富、可显多峰，缺点是较复杂、要求高。
- **变体**：分边小提琴图、叠加散点的小提琴图。

### 脊线图（同一维度几个数据分布情况）

<img src="/数据可视化/ridgeline-plot.png" width="50%">

- **基础信息**：把多组密度曲线沿纵向错落叠放，像连绵山脊，又称山脊图。

- **视觉通道**：位置颜色和形状

- **适用场景**：展示分布随时间或类别的演变，如各月温度分布、各地区收入分布。

- **注意**：连续变量，不适合较小起伏波动，权值差异不能过大，面积大小适当，要连贯的定序数据
- **优缺点**：优点是节省空间，缺点是有遮挡、不便精确读数。
- **变体**：渐变着色脊线图。

# 四、时间趋势图

### 折线图

<img src="/数据可视化/line-chart.png" width="50%">

- **基础信息**：把按时间排列的数据点用线段连接，表现数值随时间变化。

- **视觉通道**：类别→颜色；数值→坐标轴位置；

- **适用场景**：连续时间序列，如股价、流量、温度变化。

- **注意**：使用不同颜色的实线，不要虚线，所有折线最大最小值差异不要过大，横轴密度适当
- **优缺点**：优点是趋势清晰、可多线对比，缺点是不适合类别数据。
- **变体**：多系列折线图、阶梯图、光滑折线图、带置信区间折线图。

### 面积图（趋势大小）

<img src="/数据可视化/area-chart.png" width="50%">

- **基础信息**：在折线图下方填充颜色，用面积强调累积总量及变化。

- **视觉通道**：数值→平面/坐标轴视觉通道；类别→颜色视觉通道。

- **适用场景**：强调随时间累积的规模感，或堆叠表现构成随时间变化。

- **注意**：堆叠时上层基线起伏难精确读数。可加性，不得有负值。2到3组，比较同一分类数据
- **优缺点**：优点是总量感强，缺点是多系列上层不易读、易遮挡。
- **变体**：堆叠面积图、百分比堆叠面积图、河流图。

### 地平线图（实践序列可视化节约纵向空间 数据基于某个基准值变化趋势）

<img src="/数据可视化/horizon-chart.png" width="50%">

- **基础信息**：为节省纵向空间设计的时间序列图，将面积按数值带分层折叠、用颜色深浅表示高低。

- **视觉通道**：变量→位置；区分→颜色 ；幅度->位置

- **适用场景**：一屏内对比很多条时间序列，如众多传感器或股票走势。

- **注意**：有阅读门槛、需理解折叠与配色
- **优缺点**：优点是极省空间、适合密集多序列，缺点是不直观、不便精确读数。
- **变体**：双向地平线图（区分正负）。

### 河流图 (不同时间段多分类值叠加)

<img src="/数据可视化/streamgraph.png" width="50%">

- **基础信息**：居中堆叠的面积图，各系列围绕中心轴上下展开，像流动的河，又称主题河流图。

- **视觉通道**：时间→水平位置；类别→颜色视觉通道；数值→条带厚度+垂直位置。

- **适用场景**：展示多类别此消彼长和总体演变，如各话题热度随时间变化。

- **注意**：基线浮动难精确读数、类别多显杂乱
- **优缺点**：优点是有动感、表现趋势与构成演变，缺点是读数不准。
- **变体**：堆叠面积图、流图（streamgraph）。

### 瀑布图

<img src="/数据可视化/waterfall-chart.png" width="50%">

- **基础信息**：用一系列悬浮柱子表现初始值经多次增减到达终值的过程，每柱接上一柱终点。

- **视觉通道**：顺序→位置；占值→长度；增减→颜色（正值/负值）。

- **适用场景**：财务分析，如从营收到净利润的逐项增减、预算构成变化，累计效应。

- **注意**：要区分增、减和小计，正负配色一致
- **优缺点**：优点是把累积过程拆清、便于归因，缺点是只适合递推累积。
- **变体**：堆叠瀑布图、带连接线瀑布图。

### 烛形图

<img src="/数据可视化/candlestick-chart.png" width="50%">

- **基础信息**：金融行情专用，每根蜡烛表示一周期的开、收、最高、最低价，又称 K 线图。

- **视觉通道**：时间→水平位置；价格区间→垂直位置+长度；涨跌→颜色视觉通道（红涨绿跌/空心实心）。

- **适用场景**：股票、期货等价格分析的标准图表。

- **注意**：构成要开盘，收盘，最高，最低价格，多组数据不得差异过大、需专业背景、配色含义要约定
- **优缺点**：优点是信息量大、便于读盘，缺点是普通读者门槛高。
- **变体**：OHLC 条形图、带均线和成交量的组合图。

# 五、地理特征

### 分级地图

<img src="/数据可视化/choropleth-map.png" width="50%">

- **基础信息**：按行政区划给区域填色，用颜色深浅表示各区域数值，又称等值区域图。

- **视觉通道**：区域→地理位置；数值→颜色深浅（亮度/饱和度）。

- **适用场景**：指标在空间上的分布，如各省人口密度、各地区销售额。

- **注意**：受区域面积影响大、宜用密度或比率
- **优缺点**：优点是地理分布一目了然，缺点是大区域视觉权重过高、绝对量易误导。
- **变体**：双变量分级地图、分级符号地图。

### 蜂窝热力图

<img src="/数据可视化/hexbin-map.png" width="50%">

- **基础信息**：把地理空间划成等大六边形网格，用颜色表示落入其中的密度或数值。

- **视觉通道**：空间位置→网格位置；密度/数值→颜色深浅（亮度/饱和度）。

- **适用场景**：表现点数据的空间聚集，如打车热点、签到分布。

- **注意**：六边形大小要选当，过大丢细节、过小稀疏
- **优缺点**：优点是规避面积偏差、呈现密度聚集，缺点是失去精确边界。
- **变体**：方格网热力图、核密度热力图。

### 变形地图

<img src="/数据可视化/cartogram.png" width="50%">

- **基础信息**：按某数值有意改变各区域的大小或形状，让面积直接代表数值，又称统计示意地图。

- **视觉通道**：区域→地理位置；数值→面积（变形程度）。

- **适用场景**：强调数值本身而非真实面积，纠正"大区域抢眼"问题。

- **注意**：区域扭曲偏离真实形态、需适应
- **优缺点**：优点是数值直接可比、纠正面积偏差，缺点是地理形状失真。
- **变体**：连续变形地图、马赛克网格地图。

# 六、相关性

### 散点图

<img src="/数据可视化/scatter-plot.png" width="50%">

- **基础信息**：每条数据用一个点画在两变量构成的平面上，通过分布观察两变量关系。

- **视觉通道**：变量一→水平位置；变量二→垂直位置；可附加类别→颜色视觉通道。

- **适用场景**：研究相关性，看正相关、负相关、聚类和离群点，如身高与体重。

- **注意**：点多会重叠，可用透明度、抖动或采样
- **优缺点**：优点是揭示关系直观、能发现异常，缺点是只表现两变量。
- **变体**：气泡图、带拟合线散点图、散点矩阵、分面散点图。

### 气泡图

<img src="/数据可视化/bubble-chart.png" width="50%">

- **基础信息**：在散点图基础上用点的大小表示第三个变量，可再用颜色加第四维。

- **视觉通道**：变量一→水平位置；变量二→垂直位置；第三变量→面积；可附加类别→颜色视觉通道。

- **适用场景**：关系之外还想表达规模，如各国人均收入、预期寿命与人口规模。

- **注意**：面积比较不精确、应按面积映射、大气泡会遮挡
- **优缺点**：优点是维度丰富，缺点是面积读数不准。
- **变体**：动态气泡图、分面气泡图。

### 相关图

<img src="/数据可视化/correlogram.png" width="50%">

- **基础信息**：用矩阵展示多变量两两之间的相关系数，又称相关系数矩阵图。

- **视觉通道**：变量对→矩阵位置；相关强度→颜色深浅+图形大小；正负→颜色视觉通道（双色渐变）。

- **适用场景**：多变量分析中快速筛查相关变量，建模前的特征分析。

- **注意**：反映统计相关非因果、只衡量线性
- **优缺点**：优点是一次概览大量关系，缺点是不能说明因果、变量多时格子小。
- **变体**：圆点相关图、聚类相关热图。

### 热力图

<img src="/数据可视化/heatmap.png" width="50%">

- **基础信息**：用颜色矩阵表示二维表格各单元格数值，颜色越深越大。

- **视觉通道**：行列类别→位置；数值→颜色深浅（亮度/饱和度）。

- **适用场景**：两类别维度交叉下的数值分布，如各时段各星期活跃度、基因表达矩阵。

- **注意**：色阶选择和归一化关键、可配数值标注
- **优缺点**：优点是密集矩阵里快速发现热点，缺点是颜色精度有限、对色盲不友好。
- **变体**：聚类热力图、日历热力图、注释热力图。

# 七、网络关系

### 网络图

<img src="/数据可视化/network-graph.png" width="50%">

- **基础信息**：用节点表示对象、连线表示关系，展现整个关系网络结构。

- **视觉通道**：对象→节点位置+大小+颜色视觉通道；关系→连线；关系强度→连线粗细/颜色。

- **适用场景**：社交关系、知识图谱、依赖关系等无固定层级的关联。

- **注意**：节点多易成"毛线团"、需布局与交互
- **优缺点**：优点是直接表达复杂关系、发现关键节点，缺点是规模大时杂乱。
- **变体**：有向网络图、力导向图、分群网络图。

### 弧形图

<img src="/数据可视化/arc-diagram.png" width="50%">

- **基础信息**：把节点排在一条直线上，用上方或下方弧线连接有关系的节点，是网络图的线性简化。

- **视觉通道**：对象→轴上位置+颜色视觉通道；关系→弧线；关系强度→弧线粗细。

- **适用场景**：节点有自然顺序时展示连接，如人物关系、序列间关联。

- **注意**：节点排序影响疏密和美观、连线多会交叠
- **优缺点**：优点是结构清晰整洁，缺点是只适合中等规模。
- **变体**：双向弧形图、带权重弧形图。

### 和弦图

<img src="/数据可视化/chord-diagram.png" width="50%">

- **基础信息**：把各类别排在圆环上，用圆内带状弧连接，带宽表示关系强弱。

- **视觉通道**：类别→环上位置+颜色视觉通道；关系强度→带宽（弧带宽度）。

- **适用场景**：多对象间双向流动或关联，如国家间贸易、人群迁徙、跳转关系。

- **注意**：类别多时弧带密集、配色排序要精心
- **优缺点**：优点是美观、表现双向关系和强度，缺点是读数不精确。
- **变体**：有向和弦图、分组和弦图。

### 桑基图

<img src="/数据可视化/sankey-diagram.png" width="50%">

- **基础信息**：用宽度不等的带状流表现数量从一组节点流向另一组，带越宽流量越大、流量守恒。

- **视觉通道**：类别→节点位置+颜色视觉通道；流量→带宽；流向→位置方向。

- **适用场景**：能量、资金、用户在多阶段间的流转分配，如能源流向、转化路径、预算分配。

- **注意**：节点多时交叉混乱、需合理排序
- **优缺点**：优点是流向流量一目了然、表现多级分流汇聚，缺点是细小流量不易比。
- **变体**：能量流图、漏斗式桑基图、时间桑基图。

# 八、一些图像区别

### 柱形图与直方图

外形最像但本质不同。柱形图比较离散类别，横轴是互不相连的分类，柱间有间隙、可任意排序，高度表示某类别的数值；直方图描述连续变量的分布，横轴是连续数值区间，柱子紧挨无间隙、顺序不可乱，高度表示某区间内的频数。一个用来"比大小"，一个用来"看分布"。

### 饼图与玫瑰图

饼图用扇形角度表示占比，所有扇形半径相同，只靠角度区分大小；玫瑰图（南丁格尔玫瑰图）把半径也用来表示数值，角度往往相等而半径不同，靠半径甚至面积比较。饼图强调"占整体的比例"，玫瑰图更像绕成一圈的柱状图、强调数值对比，且面积编码会放大差异、更易误导。

### 堆叠面积图与堆叠柱状图

两者都用堆叠表现构成随某维度的变化，区别在横轴和侧重点。堆叠面积图横轴是连续时间，用填充面积强调总量和构成的连续演变，适合看趋势；堆叠柱状图横轴通常是离散类别或离散时间点，用一根根柱子表示每个点的构成，适合在几个节点上做比较。想看"随时间平滑变化"用前者，想看"几个节点上的构成对比"用后者。

### 其他常见区分

散点图与气泡图的区别在于气泡图多用点的大小表达第三个变量；折线图与面积图的区别在于面积图填充了线下区域、更强调总量；环形图与饼图的区别只是中心是否挖空；分组柱状图与堆叠柱状图的区别在于前者并排便于逐项比较、后者上下堆叠便于看总量和构成。选型时先想清楚要回答的是比较、构成、分布、趋势还是关系，再据此挑图，而不是凭外观相似套用。
